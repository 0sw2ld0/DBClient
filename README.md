# DBClient

## Requirements

- Xcode 8
- Swift 3
- iOS 9+

## Installation

### Cocoapods

There're 3 podspecs:

Core, common classes for any database:

```ruby
pod 'DBClient', '~> 0.3'
```

Wrapper for CoreData:

```ruby
pod 'DBClient/CoreData', '~> 0.3'
```

Wrapper for Realm:

```ruby
pod 'DBClient/Realm', '~> 0.3'
```
